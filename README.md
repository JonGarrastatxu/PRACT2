# Incendios forestales en España (1968–2016)

Aplicación Shiny para la visualización interactiva de incendios forestales en España entre 1968 y 2016.

## Contenido
- Evolución temporal de la superficie quemada
- Distribución espacial de grandes incendios
- Relación entre meteorología y severidad
- Evaluación de la eficacia operativa de los recursos de extinción

En R:
